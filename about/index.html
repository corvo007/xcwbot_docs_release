# 关于

小仓唯Bot是基于HoshinoBot与yobot开发的数个功能相异的机器人，具有功能丰富、高可用性、高安全性、高性价比等优点。
本Bot开发、部署、运营之中用到的开源免费项目/程序请见致谢小节。 因使用的部分开源项目采用的AGPL v3.0协议，且本项目已经通过因特网向用户提供了服务，因此应合规性要求，公开本项目的源码:

Github - xcwbot (opens new window)
