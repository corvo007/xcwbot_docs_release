# 2022-1-1

【会战管理系统更新公告】

我们计划于2022年1月1日22:00对会战管理系统进行更新，更新之后将全面使用新版的会战管理系统，而不再保留原有的旧系统。
新版会战系统使用指南及功能介绍：会战管理v2使用指南

Q&A

  1. 新的会战系统会继承原有会战系统的数据吗？
    不会，你需要重新建立公会。此外，如果你需要保留之前的数据，请在更新之前进行导出。
  2. 新的会战系统和原有的会战系统操作上有哪些差别？
    请参阅会战管理v2使用指南以了解。
  3. 原有的会战系统是否可以继续使用？
    不可以。
  4. 依托于原有会战系统的功能（如筛刀、合刀计算等）还能保留吗？
    可以，还能继续使用。这部分功能已经进行了重写以适配新版会战系统。
    具体指令说明请见实用功能。
    注：会战报告/离职报告已集成到网页面板统计页中，请自行前往查阅。有必要时可以导出数据以便日后查看。
  5. 会战版的会战管理系统也会同步更新吗？
    因技术原因，会战版(零号机)暂不更新，如需使用此功能请使用托管模式。
    旧版会战管理系统(yobot)指令参考：会战管理v1(yobot)指令参考
  6. 我需要做些什么吗？
    什么都不需要做，只需要等待更新完毕即可。

# 2020-11-01

分拆功能，重组Bot架构

以下为详细说明:

【小仓唯Bot功能重组公告】

根据调查结果以及多方面的考虑，我们决定将现有的Bot拆分并重组为以下四个版本：

  1. 会战版： (由原轻量版迁移)
    原轻量版，未做任何变更，仅重新命名，不设人数限制。

  2. 会战&实用版：(由原标准版迁移)
    在标准版的基础上增加了许多实用的功能，并删除了部分娱乐功能以保证功能的纯粹性，不设人数限制。

  3. 娱乐版： (由原豪华版迁移)
    本版本只包含娱乐功能，不含会战与实用功能，暂设150人为人数上限。（setu功能暂时未剥离，待观察实际情况）

  4. 终极版：
    会战&实用版与娱乐版的优惠组合套餐，即本版本包含两个Bot。

因为对原来的部分版本进行了功能调整，因此这些版本会进行重新定价，请知悉。

此外，原来三个版本的用户的迁移方案如下：

  1. 轻量版：版本改为会战版。对原轻量版用户没有影响，定价也不变。

  2. 标准版：版本改为会战&实用版。原标准版用户可使用新版本至过期，之后可再享受一个月的优惠续费价（12元/月），后恢复新定价（14.99元/月）。

  3. 豪华版：版本改为终极版。原Bot将仅包含娱乐功能。如需会战功能，须手动邀请新的会战Bot(小仓唯初号机(2077705029))入群，如不需要可以不邀请，定价不变。
    注：新的会战Bot拥有之前老版本的所有会战数据，不必担心数据丢失。

最后，计划实施版本迁移的时间为：
2020年11月1日 14:00-16:00
涉及到所有版本。
